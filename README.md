# ToDo iOS App (Assignment)

## Overview

This project is an iOS To-Do application intended to use UIKit, Storyboard, and Core Data.

## Planned Features

* Add task using a text field
* Display tasks in a table view
* Edit tasks using a dialog
* Delete tasks using swipe actions
* Save tasks using Core Data

## Current Progress

* GitHub repository created
* Initial Swift files uploaded
* Core Data work started
* Storyboard setup started in Xcode
* Basic project structure created

## Notes

Due to severe macOS/Xcode performance and memory issues during development, the full functionality could not be completed and uploaded in time.

This repository contains the work completed so far toward the assignment requirements.
